## Run locally

```powershell
cd C:\Users\sohai\Desktop\portfolio
npx serve .
```

Open the URL shown (e.g. `http://localhost:3000`), or use **Live Server** in Cursor on `index.html`.

## Edit

| What | File |
|------|------|
| Content & structure | `index.html` |
| Look & layout | `css/portfolio.css` |
| Nav, scroll, animations | `js/portfolio.js` |
| Photo | `images/sohaib.jpg` |

Update project cards in `index.html` with your real GitHub repo URLs.

## Deploy (GitHub Pages)

1. Push to your GitHub repo.
2. **Settings → Pages →** branch `main`, folder `/ (root)`.
3. Site updates in 1–2 minutes.

## Old template files

The previous Colorlib-style assets (`css/style.css`, `js/main.js`, Bootstrap, jQuery, etc.) are no longer used by `index.html`. You can delete them later to shrink the repo.
